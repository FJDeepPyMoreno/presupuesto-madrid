1. Descargar datos

El primer paso es descargar los ficheros de datos del portal de Datos Abiertos de Madrid. Para ello, introduce la URL de la página con la información de ejecución del año en curso, y específica a qué mes corresponden los últimos datos (ya que el Portal actualmente no lo indica). Al pulsar el botón 'Descargar', los datos se bajarán del Portal en unos segundos y se guardarán en el servidor, listos para el siguiente paso.

Descargando datos

2. Revisar datos

Una vez descargados, podemos revisar los datos para comprobar que las eliminaciones (flujos internos entre organismos del Ayuntamiento de Madrid) coinciden con lo esperado.

Revisando datos

3. Cargar datos

Finalmente, podemos cargar los datos descargados en la base de datos, haciéndolos visibles al exterior, tanto en español como en inglés. El proceso de carga incluye datos de ingresos, de gastos y de inversiones, y tarda unos 4-5 minutos.
