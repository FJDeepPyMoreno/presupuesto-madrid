1. Descargar datos

El primer paso es descargar los ficheros de datos del portal de Datos Abiertos de Madrid. Para ello, especifica el año de los datos que quieres descargar (por defecto estamos utilizando el ejercicio anterior al ejercicio en curso). Al pulsar el botón 'Descargar', los datos se bajarán del Portal y se guardarán en el servidor, listos para el siguiente paso.

2. Revisar datos

Una vez descargados, podemos revisar los datos para comprobar que los importes coinciden con lo esperado.

Revisando datos

3. Cargar datos

Finalmente, podemos cargar los datos descargados en la base de datos, haciéndolos visibles al exterior, tanto en español como en inglés. El proceso de carga tarda unos 4-5 minutos.
